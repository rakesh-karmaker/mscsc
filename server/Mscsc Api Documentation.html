Science Club API Documentation

Overview

This API is designed to manage the backend of a science club website. It includes user authentication, CRUD operations for managing executives, event blog posts, and article posts.

Api Link : Click Here

Authentication System

Register User

URL: /api/auth/register

Method: POST

Description: Registers a new user.

{ "name": "string", "email": "string", "password": "string" }

Login User

URL: /api/auth/login

Method: POST

Description: Logs in a user and returns a JWT token.

{ "email": "string", "password": "string" }

Executive Management

Get All Executives

URL: /api/executives

Method: GET

Description: Fetches a list of all executives.

[ { "_id": "string", "name": "string", "about": "string", "socialLinks": ["string"], "image": "string" } ]

Event Blog Management

Get All Events

URL: /api/events

Method: GET

Description: Fetches a list of all events.

[ { "_id": "string", "heading": "string", "details": "string", "workshopTag": "string", "workshopDate": "string", "coverImage": "string" } ]

Environment Variables

Variable Description
PORT Port number for the server
MONGO_URI MongoDB connection string
JWT_SECRET Secret key for JWT
CLOUDINARY_NAME Cloudinary account name
CLOUDINARY_API_KEY Cloudinary API key
CLOUDINARY_API_SECRET Cloudinary API secret

Installation

Follow these steps to set up the project:

  1. Clone the repository: git clone https://github.com/your-repo/science-club-api.git
  2. Install dependencies: npm install
  3. Configure environment variables in the .env file.
  4. Start the server: npm start
